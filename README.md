ls232-mipsL4-study
==================

这是一个用来自学mipsL4微内核技术的项目，希望最终能够运行于龙芯1B开发板上。

1 关于mipsL4微内核
------------------

[mipsL4](http://www.cse.unsw.edu.au/~disy/L4/MIPS/)是现有L4微内核中唯一一个可运行于mips架构的L4微内核实现。不过该项目已经停止开发。

2 关于龙芯1B
------------------

[龙芯1B](http://www.loongson.cn/product_info.php?id=42)是一款MIPS32指令集的嵌入式处理器。
